
Requirements
------------
Drupal 6.x

Installation
------------
1. Copy the entire bracket directory the Drupal modules directory.

2. Login as an administrator. Enable the module in admin/build/modules

Upgrading from any previous version
-----------------------------------
1. Copy the entire bracket directory to the Drupal modules directory.

2. Login as the FIRST user or change the $access_check in upgrade.php to FALSE

3. Run upgrade.php (at http://www.yourdrupalsite/update.php)

4. You may also need to clear cached data in order for all functions to work correctly
